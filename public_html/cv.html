    My App Details

    As an amateur photographer, I would like to be able to save certain locations to revisit. I have found a few apps that do this job, such as Google Maps, however these apps also do other things like tell you what is nearby which I believe make theses apps more bloated and complicated than the need to be.
    Basic Idea:
    The app I proposed to build would allow the user save their favourite photography spots when out on location. The app will only use GPS for finding the user’s location, which is an advantage in cases where location data communication cannot be accessed. For example in the Trossachs National Park there is very little 2g or 3g mobile coverage.
    Google Maps API is integrated into the app which allows the user to easily get directions back to the exact location.
    I am contining to develope this app and hopefully will release it to the Play Store.
